# Speed Probe 用户手册

## 1. 系统简介
欢迎使用 Speed Probe。这是一个专业的网络测速模拟与访客信息管理系统。您可以使用它向用户提供测速服务（或伪装成测速服务），并在后台查看详尽的访问数据。

## 2. 前台使用指南 (面向访客)
*   **访问地址**: 直接访问部署后的域名或 IP (例如 `http://your-domain.com`)。
*   **开始测速**: 页面加载完毕后，点击屏幕中央巨大的 **"开始全面测速"** 按钮。
*   **查看结果**: 系统将播放逼真的测速动画，模拟连接全球节点、测试延迟、抖动、下载和上传速度。
    *   *注意：这只是一个模拟动画，旨在提供视觉反馈。真实的信息采集在页面加载时已静默完成。*

## 3. 后台使用指南 (面向管理员)
*   **访问地址**: 访问 `/admin.php` (例如 `http://your-domain.com/admin.php`)。
*   **仪表盘**:
    *   **总访问量**: 历史累计访问次数。
    *   **今日访问**: 当天新增的访问次数。
*   **访客列表管理**:
    *   **查看列表**: 默认显示最新的访客记录。列表中包含 IP、简要位置、设备信息概览。
    *   **搜索**: 在顶部搜索框输入 IP 地址、城市名称或您备注过的内容，按回车筛选。
    *   **刷新**: 点击右上角"刷新"按钮获取最新数据。
*   **查看详情**:
    *   点击列表右侧的 **"详情"** 按钮，打开详细信息弹窗。
    *   这里可以看到 CPU 核心、GPU 型号、屏幕色深、精确的网络类型等高级指纹信息。
    *   *提示：留意弹窗底部的数据准确性说明。*
*   **添加备注**:
    *   点击 **"备注"** 按钮。
    *   在弹窗中输入文字（例如 "目标A", "公司测试机"），点击保存。备注将高亮显示在列表中，方便识别。

## 4. 移动端使用
系统针对手机和平板进行了深度优化：
*   在手机上访问后台，菜单和表格会自动适应屏幕宽度。
*   表格支持横向滑动查看更多列。

## 5. 常见问答 (Q&A)

**Q: 为什么显示的系统版本不对？**
A: 这是正常的。Chrome 等现代浏览器为了保护隐私，故意不再报告真实的 macOS 版本号（通常固定显示为 10.15.7）。

**Q: 为什么没有获取到城市信息？**
A: 如果您是在本地 (`localhost`) 测试，IP 地址是内网地址，无法进行地理定位。部署到公网服务器后会自动生效。
