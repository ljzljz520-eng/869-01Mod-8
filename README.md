# Speed Probe 网络测速探针系统

一个伪装成网络测速工具的高级访客信息采集系统。前台提供逼真的网络测速体验，实际上在后台采集详细的设备、网络和地理位置信息。

## 🎯 功能特性

*   **伪装测速**：由 Tailwind CSS 构建的现代化测速界面，包含真实的动画效果和 Ping/Jitter/Download/Upload 模拟测试。
*   **深度采集**：
    *   **基础信息**：IP地址、User-Agent、来源页面（支持识别站内/外/直接访问）。
    *   **设备指纹**：浏览器检测、操作系统识别（含 Client Hints 增强）、屏幕分辨率、色彩深度。
    *   **硬件探测**：CPU 核心数、设备内存、触控点数、WebGL 显卡信息（厂商/渲染器）。
    *   **网络诊断**：网络连接类型（WiFi/4G/有线）、下行链路速度、RTT 延迟、IP 地理位置（国家/城市/ISP）。
*   **管理后台**：
    *   实时访客列表，支持分页和关键词搜索。
    *   今日访问量与总访问量实时统计。
    *   访客详情深度查看，并将所有技术参数本地化为中文。
    *   支持为特定 IP 或访客添加备注信息。
    *   **响应式设计**：完美适配 PC、平板和手机端访问。
*   **容器化**：基于 Docker Compose 的一键部署，开箱即用。

## 🛠️ 技术栈

*   **前端**：HTML5, JavaScript (ES6+), Tailwind CSS (CDN), Vue 3 (CDN for Admin)
*   **后端**：PHP 8.2 (FPM)
*   **数据库**：SQLite 3 (无需配置 MySQL)
*   **服务器**：Nginx (作为反向代理)
*   **容器化**：Docker Desktop, Docker Compose

## 🚀 快速开始

### 1. 启动服务

在项目根目录下执行：

```bash
docker compose up -d
```

### 2. 访问系统

*   **前台测速页**：[http://localhost:3000](http://localhost:3000)
    *   发送此链接给目标用户。页面加载完成后即自动采集信息，点击"开始测速"只是视觉效果。
*   **管理后台**：[http://localhost:3000/admin.php](http://localhost:3000/admin.php)
    *   查看所有采集到的访客信息。

### 3. 停止服务

```bash
docker compose down
```

## ⚠️ 数据准确性说明

由于现代浏览器的隐私保护机制（Privacy Sandbox），部分采集数据可能存在偏差，这是正常现象：

1.  **系统版本**：Chrome 90+ 会冻结 User-Agent 中的 macOS 版本为 `10.15.7`。
2.  **设备内存**：`navigator.deviceMemory` 仅返回模糊值（如 2, 4, 8 GB）。
3.  **网络类型**：桌面浏览器通常不返回具体的物理连接类型（WiFi/Ethernet）。
4.  **IP 定位**：本地开发环境（localhost/192.168.x.x）无法获取地理位置，只有公网 IP 才能精确定位。
